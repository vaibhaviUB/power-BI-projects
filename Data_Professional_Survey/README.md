
# Data Professional Survey Breakdown (Power BI)

## Project Overview
This Power BI project analyzes survey responses from data professionals to understand
salary trends, job roles, programming language preferences, and job satisfaction levels.

## Dataset
- Survey data of data professionals from multiple countries
- Contains information about job roles, salary, age, skills, and satisfaction scores

## KPIs
- Total Survey Takers: 630  
- Average Age: 29.87  
- Work-Life Balance Score: 5.74 / 10  
- Salary Satisfaction Score: 4.27 / 10  

## Visuals
- Country-wise distribution
- Average salary by job role
- Programming language preferences
- Job difficulty and satisfaction levels

## Tools Used
- Power BI
